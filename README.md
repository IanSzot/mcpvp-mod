##The MCPVP Mod

####Version

- Forge: 1180
- MC: 1.7.10


#### Install

1. Install Forge and switch launcher to Forge profile
 - Run to verify you succesfully installed Forge
2. Move the downloaded mod file to `.minecraft\mods\`
 - Optional: move OptiFine to the same location.
